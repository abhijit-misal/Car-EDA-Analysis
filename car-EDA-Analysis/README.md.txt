# Car Dataset – Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a car dataset
to understand pricing, fuel efficiency, and engine performance.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Key Analysis
- Data cleaning and missing value handling
- Region (Origin) wise analysis
- Price (MSRP) analysis
- Fuel efficiency comparison
- Engine and performance analysis

## Key Insights
- European cars are more expensive on average
- Asian cars are more fuel efficient
- Higher cylinders reduce mileage

## Files
- Car_EDA.ipynb → Main notebook
- cars.csv → Dataset
